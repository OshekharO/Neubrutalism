# Neubrutalism CSS

Neubrutalism is a modern, minimalist CSS library that embraces bold design principles, high-contrast aesthetics, and a no-nonsense approach to web design. Inspired by the Brutalist architecture movement, this library focuses on raw, honest, and functional design.

## ğŸŒ™ Dark Mode

The library supports dark mode in two ways:

### Automatic (System Preference)
Dark mode is automatically applied when the user's system preference is set to dark mode.

### Manual Toggle
Add the `nb-dark` class to the `<html>` or `<body>` element to enable dark mode:

```html
<body class="nb-dark">
  <!-- Your content -->
</body>
```
